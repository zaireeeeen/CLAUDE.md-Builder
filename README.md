CLAUDE.md Builder

An interactive tool to help you build a custom CLAUDE.md file — the operating manual that teaches Claude (or any LLM) exactly who you are, what you need, and how to behave.

What is CLAUDE.md?

A CLAUDE.md file is a prompt that lives in your conversation or repository. It tells Claude:
* Who you are. Your role, goals, location, devices, technical level.
* What you're building toward. Short-term and long-term objectives.
* How you want to be treated. Push back, teach, nudge, stay grounded.
* What never to do. Hard limits and non-negotiable rules.
* The Canary Rule. A technique to detect when Claude loses context of your rules.
Instead of repeating yourself every conversation, Claude reads the file once and remembers it across the entire chat.

Why does this matter?

Generic Claude prompts produce generic responses. A good CLAUDE.md transforms Claude from a tool into a thinking partner that actually knows you.
This tool adapts to who you are — developer, creative, business owner, student, or personal — and only asks relevant questions. No bloat. No time wasted.

How to use it
1. Open claude-md-builder.html in any browser.
2. Answer questions about yourself. The tool adapts based on your answers.
3. Copy the generated CLAUDE.md file.
4. Paste it into a new Claude conversation, or save it in your project root for Claude Code.
That's it.

What you'll learn
* How the Canary Method prevents context rot (when Claude forgets your rules).
* Why specific rules work better than general advice.
* How to structure goals, limits, and behavior expectations.
* How to teach an AI tool about your voice, workflow, and non-negotiables.
